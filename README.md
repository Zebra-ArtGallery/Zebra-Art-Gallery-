# Zebra Art Gallery

**Located in Addis Ababa, Ethiopia.**

Zebra Art Gallery is a specialized platform dedicated to the preservation, study, and trade of authenticated vintage art and rare antiques. Our collection focuses on high-value African and Oceanic heritage pieces, curated for serious collectors and historians.

## Features
* **Curated Catalogue:** A showcase of rare, authenticated artifacts.
* **Authentication Focus:** Every piece is backed by rigorous research and professional standards.
* **Direct Engagement:** Integrated WhatsApp communication for private inquiries and high-net-worth collector services.

## Technical Details
This website is built using:
* **HTML5** for structure.
* **CSS3** for a clean, gallery-focused design.
* **JavaScript** for interactive image viewing (Lightbox).
